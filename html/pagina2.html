@mixin

La directiva @mixin se define utilizando la palabra clave @mixin seguida del nombre del mixin y una lista opcional de parámetros. El cuerpo del mixin se define entre llaves. Los parámetros se pueden pasar al mixin cuando se incluye, y se pueden utilizar dentro del cuerpo del mixin para personalizar el código generado.
Para incluir un mixin en una hoja de estilo, se utiliza la directiva @include seguida del nombre del mixin y una lista opcional de argumentos. Los argumentos se utilizan para proporcionar valores a los parámetros del mixin.
Los mixins son una herramienta poderosa para crear estilos CSS reutilizables y modulares. Pueden ayudar a reducir la redundancia en el código CSS, hacer que las hojas de estilo sean más fáciles de mantener y mejorar la consistencia del diseño.

@include

La directiva @include en Sass es una herramienta fundamental para la organización y reutilización de código CSS. Permite importar y aplicar conjuntos de estilos definidos previamente, conocidos como mixins, dentro de un archivo Sass. Esta funcionalidad ofrece diversas ventajas que facilitan el desarrollo y mantenimiento de proyectos CSS complejos.

@extend

La directiva @extend en Sass es una herramienta poderosa que permite reutilizar y compartir estilos entre diferentes selectores de CSS. Esto se logra extendiendo las propiedades de un selector a otro, lo que resulta en un código más conciso, organizado y fácil de mantener.
